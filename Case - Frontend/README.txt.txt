В файле data.js - находится данные, необходимые для решения поставленной задачи.

В файле index.html - подключение скриптов и базовая разметка страницы.

В файле scripts.js - обработка данных из файла data.js с помощью фреймворка, 
благодаря которому данные преобразуются в графики, удобные для пользователя.
